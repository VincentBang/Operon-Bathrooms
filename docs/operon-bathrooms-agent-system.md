# Operon Bathrooms Agent System

This file defines internal planning roles only. It does not implement AI agents in code.

| Agent | Purpose | Owned files/pages | Inputs | Outputs | Risk controls | When to activate | What not to do |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Project Architect Agent | Keep roadmap, repo boundaries and phase order coherent. | `docs/operon-bathrooms-master-plan.md` | Business plan, phase prompts, repo state. | Roadmap and implementation scope. | Blocks cross-repo edits and premature products. | Any new phase. | Build features outside approved phase. |
| SEO Strategy Agent | Own keyword sequencing and page priorities. | SEO plan, sitemap, Phase 1 pages. | SEO research PDF, Search Console later. | Keyword map and page targets. | Avoids broad/cheap/bad-fit traffic. | SEO planning or crawl fixes. | Create thin suburb pages. |
| Technical SEO Agent | Check metadata, canonicals, robots and sitemap. | `app/sitemap.ts`, `app/robots.ts`, metadata. | Route list, crawl results. | Crawlable public SEO surface. | Excludes admin/API/debug routes. | Before QA and release. | Index empty placeholders. |
| Content Brief Agent | Maintain page briefs and safe language. | `docs/operon-bathrooms-content-briefs.md` | SEO research, public rules. | H1/title/meta/section briefs. | Blocks final quote/legal claims. | Before content builds. | Write legal advice. |
| Competitor Intelligence Agent | Track competitor gaps. | SEO plan docs. | Competitor notes from research. | Differentiation notes. | No copied content. | Quarterly SEO review. | Chase fixed-price guarantee positioning. |
| Conversion Architecture Agent | Shape CTA paths. | Home, quote result, service pages. | Funnel goals, analytics. | CTA map and page paths. | Keeps CTAs planning-only. | Page builds and audits. | Add CRM/payment/SaaS flows. |
| Compliance/Risk Agent | Guard compliance prompts. | Public pages and quote outputs. | NSW prompt requirements, project risk fields. | Safe warnings and prompts. | No legal advice or guaranteed compliance. | Any page mentioning rules. | Tell users what is legally sufficient. |
| Internal Linking Agent | Keep the site graph useful. | Home, cost guide, service pages, sitemap. | SEO page list. | Links between estimate, review, site measure and services. | No admin links. | SEO implementation and crawl. | Link to thin placeholder child guides. |
| Measurement Agent | Define KPI tracking. | SEO plan and future analytics docs. | UTMs, Search Console, events. | Measurement plan and KPI table. | No private scoring exposure. | After Phase 1 QA. | Build CRM. |
| QA / Anti-Leak Agent | Search for unsafe wording and leaked private logic. | App, components, docs. | Repo search, build output. | Safety findings and fixes. | Blocks rate/margin exposure. | Before final report. | Publish private rate cards. |
| Cost Guide Agent | Own cost-guide positioning. | `/bathroom-renovation-cost-sydney` | SEO cost cluster. | Planning-range education. | No fixed-price promise. | Cost page updates. | Present internal rates. |
| Quote Review Agent | Own quote review brief and public output. | `/quote/review` | Quote review cluster. | Clarity-score flow and questions. | Does not judge builders or expose scoring internals. | Quote review page changes. | Say a builder is illegal/dishonest. |
| Apartment / Strata Agent | Own apartment and strata context. | Apartment service page, later strata guide. | Apartment/strata cluster. | Access, strata and Class 2 prompts. | No guaranteed approval/compliance. | Apartment page or guide work. | Promise approval timelines. |
| Waterproofing Agent | Own waterproofing education. | Guides hub now, future waterproofing guide. | Waterproofing keyword cluster. | Risk prompts and quote questions. | No DIY instructions. | Phase 2 guide work. | Teach DIY waterproofing. |
| PC Sum / Provisional Sum Agent | Own allowance education. | Guides hub now, future PC/provisional guide. | Quote allowance cluster. | Definitions and questions. | Avoids private pricing logic. | Phase 2 guide work. | Provide legal interpretation. |
| Service Page Agent | Own service landing pages. | `/services/*` | Service keyword clusters. | Scope, risks and CTAs. | No fixed prices or cheap positioning. | New service pages. | Create duplicated service pages. |
| Local SEO Agent | Own area strategy. | `/areas` and future local pages. | Local data and unique constraints. | Useful local content. | Blocks thin suburb pages. | Phase 3 local SEO. | Mass-generate suburb pages. |
| Lead Qualification Agent | Maintain private qualification rules. | `lib/bathroom-lead-qualification.ts`, admin APIs and tests. | Structured leads. | Private fit/risk/evidence labels. | Never public. | Lead intake and admin review changes. | Expose scoring logic. |
| Manual Review Agent | Maintain internal manual review report process. | `lib/bathroom-manual-review-report.ts`, admin report APIs and dashboard panel. | Lead evidence, qualification and human notes. | Internal review summary and safe copy helpers. | Admin-only; not a public proposal. | Manual review queue changes. | Autogenerate contract pricing. |
| Quote OS Agent | Future written-scope system. | Future Quote OS files. | Selections, trades, scope. | Structured scope records. | Site measure and trade checks required. | Phase 6 only. | Build procurement/payment marketplace. |
